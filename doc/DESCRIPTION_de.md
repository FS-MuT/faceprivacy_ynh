Face Privacy ist eine Webanwendung, die es einfach macht, Gesichter und bestimmte Bereiche auf Fotos zu verpixeln – vollständig im Browser. Die Anwendung garantiert Datenschutz, indem alle Vorgänge lokal auf dem Gerät des Nutzers durchgeführt werden, ohne dass Fotos auf externe Server hochgeladen werden.

### Funktionen

- Gesichterverpixelung
- Verpixelung benutzerdefinierter Bereiche
- Vollständig lokale Verarbeitung
- Export in verschiedenen Auflösungen
- Intuitive Benutzeroberfläche
